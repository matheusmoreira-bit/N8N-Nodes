# Omie n8n Node

Este pacote implementa um node customizado para a API Omie, com foco em contas a pagar.

## Recursos suportados
- Contas a Pagar: listar e lançar pagamento (baixar)
- Fornecedores: listar e atualizar
- Itens: listar e atualizar
- Pagamentos: listar e baixar

## Como usar
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Compile:
   ```bash
   npm run build
   ```
3. Registre o node no n8n usando a pasta `dist`.
